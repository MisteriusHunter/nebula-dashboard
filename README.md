🚀 Key Features
Dynamic Data Visualization: Fully interactive charts (Area, Bar, Pie, and Heatmaps) built with custom logic and high-fidelity tooltips to track real-time AI workloads and MRR.

Interactive Kanban Board: A functional task management system featuring Drag & Drop capabilities for agile project tracking.

Intelligent Navigation: Multi-section architecture using React state to toggle between Dashboard, Projects (Kanban), and deep-dive Analytics without page reloads.

Advanced UX/UI:

Glassmorphism Design: Modern translucent aesthetics with neon accents and blur effects.

Theme Switching: Full Dark/Light mode support with persistent user preferences.

Contextual Modals: Detail-oriented project overlays for granular information display.

Ultra-Responsive: Optimized for every screen size, from high-res 4K monitors to mobile devices, featuring a custom responsive sidebar and drawer menu.

🛠️ Tech Stack
Framework: Astro (for speed) + React (for interactivity).

Styling: Tailwind CSS with custom neon-shadow extensions.

Icons: Lucide React.

Interactive Components: React Hooks, Framer Motion, and Dnd-kit.

Deployment: Vercel.
